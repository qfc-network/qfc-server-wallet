## Quickstart

```sh
# 0. Have Node 20+ installed.
node --version

# 1. Install + build.
cd clients/approver-ts
npm install
npm run build

# 2. Generate a 32-byte signing key (ed25519 seed).
head -c 32 /dev/urandom > approver.key
chmod 600 approver.key

# 3. Register yourself on the server (see the Rust README, same payload).

# 4. Generate a webhook secret.
head -c 32 /dev/urandom | base64 > webhook.secret

# 5. Run.
node dist/index.js \
  --server https://qfc-wallet.example \
  --approver-id 01HABCDEFGHJKMNPQRSTVWXYZ0 \
  --secret-file ./approver.key \
  --scheme ed25519 \
  --webhook-secret "@./webhook.secret" \
  --interactive
```

## CLI

```
qfc-approver --help
```

| Flag | Default | Meaning |
| --- | --- | --- |
| `--listen` | `0.0.0.0:7000` | Webhook bind. |
| `--server` | required | qfc-server-wallet base URL. |
| `--approver-id` | required | Your registered ULID. |
| `--secret-file` | required | 32 raw bytes. |
| `--scheme` | `ed25519` | `ed25519` or `secp256k1`. |
| `--webhook-secret` | required | HMAC secret; `@/path` to read from a file. |
| `--auto-approve` | `false` | Approve every request. **Demo only.** |
| `--auto-reject` | `false` | Reject every request. |
| `--interactive` | `false` | Prompt on stdin. |
| `--audit-path` | `~/.qfc-approver/audit.log` | NDJSON audit destination. |

## Embedding

```ts
import { buildApp } from "qfc-approver/dist/webhook.js";
import { Processor } from "qfc-approver/dist/processor.js";

const processor = new Processor({
  server: "https://wallet.example",
  approverId: "01H...",
  policy: "interactive",
  auditPath: "/var/log/qfc/approver.log",
  scheme: "ed25519",
  secret: mySecret32Bytes,
});
const app = buildApp({
  hmacSecret: myWebhookSecret,
  handler: (req) => processor.process(req).then(() => undefined),
});
app.listen(7000);
```
